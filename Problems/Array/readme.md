## Folder Structure

Each problem is organized as follows:

```
Problems/
│
├── <Topic>/
│ ├── <ProblemName>/
│ │ ├── Solution/
│ │ │ └── Solution.java
│ │ ├── InputOutput/
│ │ │ ├── input.txt
│ │ │ └── output.txt
│ │ └── TestCases/
│ │ ├── test_case1.txt
│ │ ├── test_case2.txt
│ │ └── test_case3.txt
│ └── ...
└── ...
```
