# Problem Statement: Palindrome String

## Description

Given a string S, check if it is palindrome or not.

## Examples

### Example 1

```bash
**Input:**
S = "abba"

Output:
true

Explanation:
S is a palindrome
```

Example 2
Input:

int[] nums = {2, 1, 3, 4};
Output:

java
Copy code
false
Explanation:
There is no sorted array once rotated that can make nums.
