# Problem Statement: Palindrome String

## Description

Given a string S, check if it is palindrome or not.

## Examples

### Example 1

```bash
Input:
S = "abba"

Output:
true

Explanation:
S is a palindrome
```

Example 2

```bash
Input:
S = "abc"

Output:
true

Explanation:
S is a palindrome
```

Example 3

```bash
Input:
S = "abbcda"

Output:
false

Explanation:
S is not a palindrome
```

Constraints
1 <= Length of S<= 2\*105
